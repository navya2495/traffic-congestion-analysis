Impact of Traffic Congestion on NYC Taxi Trips

Team: Jaladurgam Navya

Project Overview

This project investigates how traffic congestion affects taxi trip durations and fares across New York City. It also explores spatial and temporal variations in congestion and applies various modeling techniques for prediction.

Methodology

  • Data Cleaning & Preprocessing
  • Descriptive, Spatial, and Temporal Analysis
  • Regression & Machine Learning (Random Forest, Decision Tree, Linear Regression)
  • Visualization with charts and maps
  • Model Evaluation using MSE, MAE, R²

Data Preprocessing

  • Converted pickup and dropoff columns to datetime
  • No duplicate rows
  • Negative values handled using absolute conversion
  • Missing values imputed using mode
  • Label encoding applied to store_and_fwd_flag
  • Excluded RatecodeID = 99 (not in data description)

Descriptive Statistics

Spatial Analysis

  • Average trip durations and fares by neighborhood
  • Computed for both pickup and dropoff areas

Temporal Analysis

  • Hourly variations in congestion levels
  • Derived from pickup hour and day of week

Correlation with Congestion Surcharge

  • Trip duration and distance: Little to no correlation
  • Total fare: Moderate positive correlation

    • Trip duration and trip distance have very little to no relationship with congestion surcharge. Total fare has a moderate positive relationship with congestion surcharge, suggesting that higher fares may be associated with slightly higher congestion surcharges, but the correlation is not very strong.

Modeling & Metrics

Linear Regression

  • High MSE and MAE
  • Weak R² score

    • MSE :Higher values indicate lower performance R- squared:Indicating a weak level of explanatory power. MAE:Higher values indicate lower accuracy.

Decision Tree Regression

  • MSE: Low
  • R²: 0.32
  • MAE: Low

    • MSE :Lower values indicate better performance An R-squared of 0.32 means that 32% of the variance in the fare amount is explained by the model.​ MAE:Lower values indicate better accuracy

Random Forest Regression

  • Best performance overall
  • Lowest error, highest R²

    • MSE:Lower values indicate better performance.​ R-squared:Indicating a moderate level of explanatory power.​ MAE:Lower values indicate better accuracy.​ The Random Forest Regression model is performing​  the best among the three models based on the provided​  metrics. It offers the lowest errors and the highest ​ explanatory power.

Predicted vs Actual Values

Compared predictions from all three models against actual values based on congestion and neighborhood. Random Forest showed the best fit.

Conclusion

  • High congestion neighborhoods: 82, 182, 18, 153, 191, 9, 31
  • These areas showed longer trip durations and higher fares
  • Random Forest provided the most accurate predictions
